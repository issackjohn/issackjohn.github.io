About

I'm a software engineer at Microsoft Edge focused on web performance and browser benchmarking. I'm a core contributor to Speedometer 3.0, the cross-browser benchmark — built by Apple, Google, Microsoft, and Mozilla — that measures how browsers handle the demands of real-world web applications.

My work centers on making the web measurably faster: profiling real workloads, designing representative benchmarks, and chasing simplicity in the process.
